Le but d'un navigateur web (Chrome, Edge, Firefox, Safari) est de lire des documents HTML et de les afficher correctement.

Un navigateur n'affiche pas les balises HTML, mais les utilisent pour déterminer comment afficher le document.

Qu'est ce que le HTML

HTML est le langage de balisage standard pour la création de pages Web. HTML signifie Hyper Text Markup Language.

On utilise le langage html pour la création de site web et application web. Le Html décrit la structure d'une page web, il se compose d'élément. Ces élément indiquent au navigateur, comment afficher le contenue.

Voici la structure d'une page web, c'est ce qu'on appelle le squelette html, il est la base de tous les pages web construisent en html.

Nous allons vous donner un peu de détails sur le squelette html

  • <!DOCTYPE html> (Type de Document) indique au navigateur que ce document est un document HTML5 , on le place toujours avant le squelette.
  • <html> est l'élément racine d'une page HTML C'est a l'intérieur de cette balise qu'on va placer la balise <head> et la balise <body>
  • <head> élément contient des méta-informations sur la page HTML, C'est a l'intérieur de cette balise qu'on trouve les information personnel de notre page
  • <title> élément spécifie un titre pour la page HTML (qui s'affiche dans la barre de titre du navigateur ou dans l'onglet de la page)
  • <body> élément définit le corps du document et est un conteneur pour tous les contenus visibles, tels que les titres, les paragraphes, les images, les hyperliens, les tableaux, les listes, etc.
